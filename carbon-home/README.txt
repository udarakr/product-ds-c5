  WSO2 Unified UI Framework (UUF)
---------------------------------------------------------

WSO2 UUF is a Unified Ui Framework for Composable Web Apps.


Running & Installation
-----------------------------
There are two modes of running WSO2 UUF. See the documentation
for more details.
http://docs.wso2.com/microservices-framework-for-java/Key+Concepts#KeyConcepts
